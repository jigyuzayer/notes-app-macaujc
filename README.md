# notes-app-macaujc

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内容以静态网页文件为主，不针对任何特定域名或网站，主要用于整理、存储和展示各类独立页面资源。

## 目录说明

```
notes-app-macaujc/
├── index.html          # 仓库首页 / 总览入口
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面，页面之间相互独立，无依赖关系。
- `assets/`：存放页面引用的静态资源，按类型分目录管理。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开查看。
- 页面内容不包含外部链接或跳转至其他网站。
- 归档页面可能包含示例数据、演示内容或技术测试页面，仅供学习与参考。
- 页面文件名建议使用英文小写、连字符分隔，以便于管理。

## 维护说明

- 如需新增页面，请在 `pages/` 下创建新的 HTML 文件，并在 `index.html` 中添加链接（如适用）。
- 静态资源请放置于 `assets/` 对应子目录，避免与页面文件混放。
- 请勿在页面中嵌入跟踪脚本、广告或任何形式的商业推广内容。
- 本仓库仅用于归档展示，不保证页面内容的长期可用性或兼容性。
- 欢迎提交 Issue 或 Pull Request 以改进页面质量或修复问题。

## 许可

本仓库内容仅供个人学习与归档使用。如涉及第三方素材，请自行确认其使用许可。
